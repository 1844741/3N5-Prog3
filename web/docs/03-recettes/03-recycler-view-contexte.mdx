# RecyclerView - Context

Dans cette recette, nous allons voir comment 

- Réagir à un clic sur un élément d'une liste dans un `RecyclerView`.
- Démarrer une nouvelle activité à partir d'un `ListAdapter`.
- Afficher un `Toast` à partir d'un `ListAdapter`.
- Modifier un élément de la liste à partir d'un `ListAdapter`.

Pour y arriver, nous allons devoir comprendre à quoi sert l'objet `Context` et comment l'utiliser.

Par défaut, le `Context` de l'activité n'est pas accessible dans un `ListAdapter`. Nous allons voir comment l'utiliser à partir d'un `ListAdapter`.
